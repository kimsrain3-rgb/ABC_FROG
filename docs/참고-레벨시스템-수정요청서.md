# ABC Frog 게임 수정 요청서

## 📋 현재 상황
- 원본 게임: 사용자가 직접 그린 PNG 캐릭터를 Base64로 인코딩하여 삽입
- 문제점: 지적된 개선 사항이 전혀 반영되지 않음
- 파일 크기: 8.6 MB (Base64 인코딩된 이미지/사운드 포함)

---

## ❌ 현재 게임의 문제점

### 1. **레벨 시스템 없음**
- 무한 반복 플레이만 가능
- 목표나 진행도 개념이 없음
- 난이도 변화 없음

### 2. **진행도 저장 없음**
- 브라우저 닫으면 모든 진행 상황 초기화
- 성취/기록이 남지 않음

### 3. **난이도 증가 없음**
- 파리 속도가 계속 동일
- 시간이 지나도 쉬워지거나 어려워지지 않음
- 지루함 발생

### 4. **명확한 목표 없음**
- 언제 끝나는지 모름
- 성취감 없음
- 별점/평가 시스템 없음

### 5. **반복 플레이 동기 부족**
- 다시 플레이할 이유가 없음
- 수집 요소 없음
- 경쟁 요소 없음

---

## ✅ 요청 사항

### **필수 구현 항목**

#### **1. 5단계 레벨 시스템**

```
Level 1: A-E (5글자)
- 파리 속도: 기본 속도
- 목표: A, B, C, D, E 파리를 모두 잡기

Level 2: F-J (5글자)
- 파리 속도: 기본 속도 × 1.3
- 목표: F, G, H, I, J 파리를 모두 잡기

Level 3: K-O (5글자)
- 파리 속도: 기본 속도 × 1.6
- 목표: K, L, M, N, O 파리를 모두 잡기

Level 4: P-T (5글자)
- 파리 속도: 기본 속도 × 2.0
- 목표: P, Q, R, S, T 파리를 모두 잡기

Level 5: U-Z (6글자)
- 파리 속도: 기본 속도 × 2.5
- 목표: U, V, W, X, Y, Z 파리를 모두 잡기
```

**핵심**: 각 레벨은 **해당 문자의 파리만** 나타나야 합니다!

---

#### **2. 레벨 선택 메뉴**

게임 시작 시:
- 5개 레벨 카드가 그리드로 표시
- 각 카드에는:
  - 레벨 번호 (Level 1, Level 2...)
  - 해당 문자 (A-E, F-J...)
  - 획득한 별 개수 (⭐⭐⭐)
  - 최고 점수
  - 잠금 상태 (Level 1만 처음부터 열림)

UI 예시:
```
┌─────────┐ ┌─────────┐ ┌─────────┐
│ Level 1 │ │ Level 2 │ │ Level 3 │
│  A - E  │ │  F - J  │ │  K - O  │
│ ⭐⭐⭐  │ │ ⭐⭐☆  │ │ 🔒 잠김 │
│ 점수:250│ │ 점수:180│ │         │
└─────────┘ └─────────┘ └─────────┘
```

---

#### **3. 레벨 완료 조건**

각 레벨의 목표 문자를 **모두 최소 1회씩** 잡으면:
1. "Level Complete!" 화면 표시
2. 별점 계산 (1~3개)
3. 다음 레벨 자동 언락
4. 버튼:
   - [다음 레벨] (다음 레벨로 이동)
   - [메뉴로] (레벨 선택 화면으로)

---

#### **4. 별점 시스템 (1~3개)**

성과에 따라 별 개수 결정:

- **⭐⭐⭐ (3성)**: 
  - 실수 0회 (틀린 파리 안 잡음)
  - 콤보 5 이상 달성
  
- **⭐⭐☆ (2성)**:
  - 실수 1~2회
  - 콤보 3 이상
  
- **⭐☆☆ (1성)**:
  - 레벨 완료 (조건 미달)

---

#### **5. LocalStorage 진행도 저장**

저장 항목:
- 각 레벨 언락 상태
- 각 레벨 최고 별점
- 각 레벨 최고 점수
- 총 획득 별 개수

브라우저 닫아도 유지되어야 함!

---

#### **6. 게임 중 UI 추가**

**상단 표시**:
- 현재 레벨 제목 (예: "Level 1: A-E")
- 진행도 바 (수집한 문자 / 목표 문자)
  - 예: 3/5 ━━━━━ (60%)

**버튼**:
- 💡 **힌트 버튼** (오른쪽 하단)
  - 클릭 시: 아직 안 잡은 문자 중 하나를 강조 (2초간 반짝임)
  
- ⬅️ **메뉴 버튼** (왼쪽 상단)
  - 클릭 시: "메뉴로 돌아가시겠습니까?" 확인 후 레벨 선택 화면으로

---

#### **7. 난이도 곡선**

| 레벨 | 파리 속도 | 파리 개수 | 스폰 간격 |
|------|----------|----------|----------|
| 1    | 1.0x     | 3~4개    | 2.0초    |
| 2    | 1.3x     | 4~5개    | 1.8초    |
| 3    | 1.6x     | 4~5개    | 1.6초    |
| 4    | 2.0x     | 5~6개    | 1.4초    |
| 5    | 2.5x     | 5~6개    | 1.2초    |

**중요**: 속도가 실제로 증가해야 합니다!

## 🎨 **디자인 유지 사항**

### ✅ **절대 변경하면 안 되는 것들**

1. **모든 그래픽 (PNG 이미지)**
   - 개구리 캐릭터
   - 파리
   - 배경 (연못, 잎사귀 등)
   - 애니메이션 스프라이트
   
2. **모든 사운드 (Base64 오디오)**
   - 배경음악
   - 효과음 (파리 잡기, 실패 등)
   
3. **기존 애니메이션**
   - 개구리 혀 나오는 애니메이션
   - 파리 날아다니는 애니메이션
   - 스타버스트 효과
   - 콤보 표시

4. **원본 게임 느낌**
   - 귀여운 분위기
   - 색감
   - 폰트 (Fredoka)

## 🔧 **구현 방법 제안**

### **원본 코드 수정 위치**

1. **파리 스폰 함수**
   - 현재: 랜덤하게 A-Z 중 아무 문자
   - 수정 후: 현재 레벨의 문자만 (예: Level 1이면 A,B,C,D,E만)

2. **파리 속도 설정**
   - 현재: 고정 속도
   - 수정 후: 레벨별 배율 적용 (speed × levelSpeedMultiplier)

3. **게임 시작 전**
   - 레벨 선택 메뉴 표시
   - 사용자가 레벨 선택 후 게임 시작

4. **파리 잡기 처리**
   - 잡은 문자를 Set에 저장
   - 목표 문자를 모두 수집했는지 체크
   - 완료 시 레벨 완료 화면 표시
