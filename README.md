# 🐸 ABC FROG - Kids English Learning Game

배고픈 개구리에게 알파벳 파리를 잡아 먹여주는 교육 게임

## 📁 폴더 구조
```
ABC_FROG/
├── index.html          ← 메인 게임 파일
├── assets/
│   ├── images/         ← 개구리, 파리, 배경 PNG
│   ├── sounds/         ← BGM, 효과음, A~Z 발음 mp3
│   └── fonts/          ← 폰트
├── versions/           ← 이전 버전 백업
├── docs/               ← 기획 문서
└── README.md
```

## 🎮 게임 특징
- 손그림 점박이 개구리 캐릭터 (77프레임 애니메이션)
- 손그림 파리 (날개 퍼덕 애니메이션)
- A~Z 알파벳 학습 + 생활영어 문장 10가지
- TTS + MP3 발음 지원
- 모바일 최적화 (dvh, safe-area)

## 🛠 기술 스택
- HTML5 + JavaScript (단일 파일)
- Base64 내장 에셋 (이미지, 사운드)
- 반응형 디자인 (PC/모바일 대응 예정)
